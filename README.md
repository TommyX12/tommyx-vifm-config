# tommyx-vifm-config

## Setup
- Install [Vifm](https://vifm.info/)
    - `brew install vifm` (macOS)
- Install optional dependencies
    - `poppler`
        - `brew install poppler` (macOS)
    - `fd`
        - `brew install fd` (macOS)
- Create symlink for config
    - `mkdir -p ~/.config/vifm`
    - `ln -s <path-to-repo>/vifmrc ~/.config/vifm/vifmrc`
